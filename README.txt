# 🧠 WordPress Image Quiz Builder Plugin

A lightweight and user-friendly WordPress plugin that lets you create image-based multiple-choice quizzes with ease. Supports 5 questions per quiz and displays image-based answers at the end. Perfect for interactive content, personality quizzes, or educational tools.

## ✨ Features

- 📸 Image-based multiple choice questions
- 📝 Supports exactly 5 questions per quiz
- 🎯 Displays a result based on selected answers (image output)
- 🧩 Easy integration with WordPress via shortcode
- 📱 Responsive and mobile-friendly
